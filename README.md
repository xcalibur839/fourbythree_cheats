# Cheat at Hank Green's 4x3 game

There really isn't a point to cheating at this game since there are no leaderboards and cheating completely removes any challenge or fun from the game itself.

Creating this was a fun little reverse engineering challenge for myself and really nothing more. If you want to use it to trick your friends for some reason, or look through the code to get an understanding of how the game records game data, history, and stats, have at it.

## Dependencies:
1.  Unix-like terminal (bash/fish/zsh/etc. on Linux/Mac, WSL on Windows)
2.  python 3
3.  wget

## First time setup:
1.  Clone this repository
2.  Run `setup.sh`

After setup is complete, run `cheat.sh` and follow the instructions whenever you want to have a perfect 4x3 score saved to your Browser.
